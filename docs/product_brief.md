# Product Brief

## Product

ActionPromptGuard

## Buyer

Small engineering teams, solo SaaS builders, dev agencies, and open-source maintainers adding AI agents or PR automation to GitHub Actions.

## Pain

AI coding agents and PR bots often get added quickly. Teams may not notice when a workflow:

- runs on `pull_request_target`,
- has write permissions,
- checks out untrusted PR code,
- passes PR text directly into prompts,
- references secrets,
- uses third-party actions pinned only to tags.

That combination can create a security mistake in the release path.

## Promise

"Scan your GitHub workflows before AI agents turn pull requests into a security boundary problem."

## Monetization

Start with a free CLI to get trust and distribution.

Paid package:

- Pro report generator.
- GitHub Action comment template.
- Policy presets for agencies and teams.
- One-page PDF report for client handoff.
- Priority rule updates.

Suggested first price:

- $19 one-time for Pro templates/report bundle.
- $79 for agency bundle with client report language.
